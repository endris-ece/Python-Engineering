# Resistor Color Code Calculator

## Description

A Python program that calculates the resistance value of a resistor based on its color bands.

The calculator supports both 4-band and 5-band resistors and determines the resistance value and tolerance.

## Features

- Supports 4-band resistors
- Supports 5-band resistors
- Color-based resistor calculation
- Input validation
- Displays resistance value with tolerance

## Concepts Practiced

- Dictionaries
- Functions
- Loops
- Exception handling
- User input validation
- String formatting

## How It Works

The user selects:

1. Resistor type:
   - 4 Bands
   - 5 Bands

2. Enters resistor band colors.

3. The program calculates the reult
