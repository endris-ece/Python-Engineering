# Logic Gate Simulator

##  Description 

A python program that calculates the output of different logic gate input combinations.
The program supports gates ["NOT", "OR", "AND", "NOR", "NAND", "XOR", "XNOR"]

## Features

- Gate based output calculation
- Supports upto 26 inputs
- Input validation
- Display gate, gate inputs with output

# Concepts practiced

- Dictionaries
- Lists
- Functions
- List Comprehension
- Loops
- Exception handling
- User input validation
- String formatting

## How it works

The user selects:

- gates:

- enter each inputs:
