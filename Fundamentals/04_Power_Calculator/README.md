# Power Calculator

## Description
A python command_line application that calculates an electrical quantity of their choice using the formulas:

P = VI
P = V²/R
P = I²R

The user chooses a quantity to calculate and provide two values of the remaining quantities.

## Features

-Calculate Voltage
-Calculate Current
-Calculate Power
-Calculate Resistance
-Input Validation
-Handles division by Zero
-Formatted Output

## Example
Choose the quantity to calculate:
 1. power
 2. voltage
 3. current
 4. resistance
