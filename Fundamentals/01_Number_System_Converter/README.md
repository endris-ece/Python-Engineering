# Number System Converter

## Description

A command-line Python application that converts numbers between Binary, Octal, Decimal, and Hexadecimal.

The program first converts the input number to decimal and then converts the decimal value to the requested target number system. All conversion algorithms are implemented manually without using Python's built-in base conversion functions.

## Features

- Convert Binary to Decimal, Octal, and Hexadecimal
- Convert Octal to Binary, Decimal, and Hexadecimal
- Convert Decimal to Binary, Octal, and Hexadecimal
- Convert Hexadecimal to Binary, Octal, and Decimal
- Validate user input
- Menu-driven command-line interface
