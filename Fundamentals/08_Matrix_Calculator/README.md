# Matrix Calculator

A command-line Matrix Calculator written in Python. This project performs common matrix operations without using external libraries such as NumPy. All calculations are implemented manually using Python lists, loops, and functions to strengthen understanding of matrix mathematics and algorithm implementation.

## Features

* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Scalar Multiplication
* Matrix Transpose
* 2×2 Matrix Determinant
* 2×2 Matrix Inverse

## Concepts Practiced

* Functions
* Nested loops
* Lists and 2D lists
* Dictionaries
* Input validation
* Matrix indexing
* Mathematical algorithms
* Problem decomposition
* Modular program design

## Skills Demonstrated

* Working with two-dimensional data structures
* Implementing matrix operations from first principles
* Validating matrix dimensions before calculations
* Separating user input, computation, and output
* Organizing large programs into reusable functions
