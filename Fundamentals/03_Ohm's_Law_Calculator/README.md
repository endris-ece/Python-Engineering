# Ohm's Law Calculator

## Description

A Python command-line application that calculates one electrical quantity using Ohm's Law.

Formula:
V = I × R

The user chooses which quantity to calculate and enters the remaining two values.

## Features

- Calculate Voltage
- Calculate Current
- Calculate Resistance
- Input validation
- Handles division by zero
- Formatted output

## Example

Select quantity:

1. Voltage
2. Current
3. Resistance

Choice: 1

Current: 2
Resistance: 15

Voltage = 30.00 Volt
